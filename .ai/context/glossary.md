# Glossary

> Domain-specific terms used in this codebase. Update as new concepts are introduced.

| Term | Definition |
|------|-----------|
| **Turbo** | The monorepo project name and npm scope (`@turbo/*`) |
| **App** | A deployable application (`apps/web`, `apps/mobile`) |
| **Package** | A shared library consumed by apps (`packages/*`) |
| **Tooling** | Shared configuration packages (`tooling/*`) |
| **shadcn/ui** | Component system using Radix primitives with Tailwind CSS styling |
| **CVA** | Class Variance Authority — utility for defining component variants |
| **Hono** | Lightweight web framework used for the API layer |
| **Better Auth** | TypeScript-first authentication library used for auth |
| **Drizzle** | TypeScript ORM for PostgreSQL |
| **Trigger.dev** | Background job/task runner platform |
